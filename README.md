# Daily Reflection Tree — Assignment Submission

This submission is part of the DeepThought Fellowship assignment to design a deterministic reflection system using a structured decision tree.

---

## 🧠 Objective

The objective of this assignment is to build a reflection tool that guides users through a structured end-of-day analysis using predefined questions and deterministic branching.

The system avoids AI at runtime and instead encodes logic directly into the tree structure.

---

## 🧩 Approach

The reflection process is designed across three psychological axes:

### 1. Agency (Locus of Control)
Focuses on how the user perceives control over events — whether outcomes are driven by their actions or external factors.

### 2. Contribution (Orientation)
Shifts attention from outcomes to contribution — whether the user added value or focused on expectations.

### 3. Perspective (Radius)
Expands reflection beyond the self — encouraging awareness of others such as team members or customers.

These axes are sequenced to create a progressive reflection flow:
Agency → Contribution → Perspective

---

## ⚙️ Design Principles

- Deterministic logic (same input → same output)  
- Fixed answer options (no free-text input)  
- Structured branching through decision nodes  
- Reflections designed to increase awareness without judgment  

---
## 🔄 How the System Works

- The user is guided through a sequence of predefined questions  
- Each answer leads to a deterministic path in the tree  
- Reflection nodes provide insights based on the chosen path  
- A final summary synthesizes the overall reflection  

---

## 📌 Notes

This submission focuses on clarity, structure, and deterministic design rather than complexity.  
The goal is to demonstrate how psychological concepts can be translated into a structured decision system.
